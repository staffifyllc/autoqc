# AutoQC.io - Vercel Deployment Guide

Step-by-step to get AutoQC.io live.

## Prerequisites

- ✅ AWS backend deployed
- ✅ API keys in .env.local (locally on your machine)
- ⏳ Vercel account
- ⏳ GitHub account (staffifyllc)
- ⏳ AutoQC.io domain

## Summary

1. Push code to GitHub (repo: staffifyllc/autoqc)
2. Import repo into Vercel
3. Add all 13 env vars from your local .env.local file
4. Deploy
5. Add autoqc.io domain
6. Set up Stripe webhook

All env var values are in your local `.env.local` file. Never commit these to git.

See the main README for setup instructions.
